# DC 数据连接器 - UI 设计规范

## 设计风格

**定位**：奶油质感编辑风（Cream Editorial）

温暖的象牙白底色，搭配柔和的暖灰层次和琥珀金点缀色，营造高端 SaaS 的精致感。避免冷冰冰的纯白，用微妙的暖色调和纸质纹理质感传递品质感。

| `--radius-md` | 12px | 数据卡片、表格容器 |
| `--radius-lg` | 16px | 弹窗 |
| `--radius-pill` | 100px | 标签、徽章、筛选按钮 |

## 动效规范

- **页面加载**：卡片依次 fadeUp 淡入（`animation-delay` 递增 0.08s）
- **数字跳动**：统计数字 count-up 动画，1.2s cubic-bezier 缓出
- **运行中脉冲**：运行中状态点 `pulseDot` 1.5s 循环呼吸
- **卡片 hover**：`translateY(-2px)` + 阴影提升，0.3s 过渡
- **弹窗弹入**：`fadeUp` 0.3s，背景 `rgba(45,42,38,0.3)` + `backdrop-filter: blur(4px)`

## 组件规范

### 状态徽章
胶囊形状（`border-radius: 100px`），配色跟随语义：
- 成功/启用：绿底绿字 `--accent-green-bg / --accent-green`
- 失败/告警：红底红字 `--accent-red-bg / --accent-red`
- 运行中：蓝底蓝字 + 脉冲点
- 等待中/暂停：琥珀底琥珀字
- 已取消：灰底灰字

### 表格
- 表头：11px 大写字母，0.8px 字距，三级灰色
- 行 hover：`--bg-hover` 背景
- 最后一行无底边框

### 卡片
- 顶部 3px 色条（渐变，颜色跟随平台/类型）
- 内边距 20-24px
- hover 提升 + 边框变暖

### 弹窗
- 居中弹出，420-440px 宽度
- 暗色半透遮罩 + 毛玻璃
- 表单标签 12.5px 500 weight
- 确认按钮铜色渐变，取消按钮灰底

### 侧边导航
- 宽度 240px，固定定位
- 当前页琥珀铜左边框 3px 指示条
- 未读徽章红底白字胶囊

## Noise 纹理
全局 `body::before` 叠加 SVG fractalNoise 纹理，`opacity: 0.018`，增加纸质质感。

## 响应式断点
- `<= 1200px`：统计卡片 2 列，图表单列，底部单列
- `<= 768px`：隐藏侧边导航，统计卡片单列
